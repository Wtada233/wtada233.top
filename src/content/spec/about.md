# 关于本博客

这是一个深度魔改的 [Fuwari](https://github.com/saicaca/fuwari) 衍生版本。

与其说它是一个博客，不如说它是一个**极致静态化工程**。我不仅在这里记录生活，更在测试如何构建一个能抵御时间侵蚀、无视服务下线、具备极强“离线生存能力”的数字空间。

### 核心特性
- **全自动化流水线**: 基于 Zod 的配置校验，Sharp 的封面主色提取。
- **数字遗产保护**: 自动化的 GitHub 静态化与外链图片本地化脚本。
- **极致性能**: 自动字体子集化压缩，无 CDN 依赖。
