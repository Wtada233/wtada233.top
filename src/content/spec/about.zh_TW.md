# 關於本部落格

這是一個深度魔改的 [Fuwari](https://github.com/saicaca/fuwari) 衍生版本。

與其說它是一個部落格，不如說它是一個**極致靜態化工程**。我不僅在這裡記錄生活，更在測試如何構建一個能抵禦時間侵蝕、無視服務下線、具備極強「離線生存能力」的數位空間。

### 核心特性
- **全自動化流水線**: 基於 Zod 的配置校驗，Sharp 的封面主色提取。
- **數位遺產保護**: 自動化的 GitHub 靜態化與外鏈圖片在地化腳本。
- **極致性能**: 自動字體子集化壓縮，無 CDN 依賴。
