# 이 블로그에 대하여

이곳은 [Fuwari](https://github.com/saicaca/fuwari)를 기반으로 깊게 커스텀된 파생 버전입니다.

단순한 블로그라기보다 **「극한의 정적화 프로젝트」**에 가깝습니다. 저는 이곳에서 일상을 기록할 뿐만 아니라, 시간의 흐름에 저항하고 외부 서비스 중단에 구애받지 않는, 강력한 "오프라인 생존 능력"을 갖춘 디지털 공간을 구축하는 방법을 테스트하고 있습니다.

### 핵심 기능
- **완전 자동화 파이프라인**: Zod 기반 설정 검증, Sharp를 이용한 커버 이미지 주요 색상 추출.
- **디지털 유산 보호**: 자동화된 GitHub 정적화 및 외부 이미지 로컬라이징 스크립트.
- **극한의 성능**: 자동 폰트 서브셋팅 압축, CDN 의존성 제로.
