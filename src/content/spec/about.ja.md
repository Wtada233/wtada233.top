# このブログについて

これは [Fuwari](https://github.com/saicaca/fuwari) をベースに大幅なカスタマイズを施した派生版です。

単なるブログというよりは、**「究極の静的化プロジェクト」** と呼ぶべきものです。ここでは日々の記録だけでなく、時間の侵食に耐え、外部サービスの停止を無視し、強力な「オフライン生存能力」を備えたデジタル空間をいかに構築するかをテストしています。

### 主な特徴
- **完全自動化パイプライン**: Zod による設定バリデーション、Sharp によるカバー画像主色抽出。
- **デジタル遺産保護**: 自動化された GitHub 静的化および外部画像ローカライズスクリプト。
- **究極のパフォーマンス**: フォントの自動サブセット化圧縮、CDN への依存ゼロ。
